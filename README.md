# Lab-3-Construire-un-cran-d-accueil-d-filant-avec-ScrollView

## TP — Transformation d’un écran fixe en écran défilant avec ScrollView (React Native + Expo Router)
### Objectif du TP

L’objectif de ce TP est de transformer un écran d’accueil statique en un écran défilant grâce au composant ScrollView de React Native.
L’écran contient un texte long qui dépasse la hauteur de l’écran, donc il doit être rendu scrollable, avec une barre de défilement blanche sur iOS pour respecter le thème visuel sombre de l’application.


### Structure du projet utilisée

Le projet utilise Expo Router et possède la structure suivante :

Un dossier app/ qui contient les écrans

Un fichier app/index.tsx qui représente l’écran d’accueil

Un fichier _layout.tsx dans app/ qui gère la navigation

Un dossier components/ .


### Etapes de réalisation 

Dans index.tsx :

L’écran d’accueil doit être modifié pour :

contenir un composant ScrollView


afficher un titre

afficher un texte long

permettre à l’utilisateur de défiler verticalement
